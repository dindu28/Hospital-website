# Hospital Management Website

A simple and responsive **Hospital Management System** built using **HTML, CSS, and JavaScript**.  
This project allows patients to book appointments, edit or delete them, and view all scheduled appointments in a dashboard.

---

## Features

- Book appointments
- Edit existing appointments
- Delete appointments
- Automatically display appointments in a dashboard
- Data stored using LocalStorage (no backend required)
- Fully responsive design (mobile-friendly)
- Clean and modern user interface

---

## Pages Included

- **Home Page** – Introduction to the hospital
- **Services Section** – Displays hospital services
- **Doctors Section** – Shows available doctors
- **Patient Portal** – Book, edit, and delete appointments
- **Dashboard** – Displays all scheduled appointments

---

## How It Works

- Appointments are stored in the browser using **LocalStorage**.
- When a user books an appointment:
  - The data is saved to LocalStorage.
  - The appointments table updates automatically.
- Users can:
  - Click **Edit** to update appointment details.
  - Click **Delete** to remove an appointment.
- The dashboard refreshes automatically to reflect updated data.

---

## Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & responsiveness
- **JavaScript** – Functionality and local storage management
